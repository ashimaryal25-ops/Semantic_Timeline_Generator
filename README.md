A Python tool that extracts **events and dates** from text and organizes them into a **chronological timeline**. The timeline is displayed interactively using **Streamlit**.

It uses **spaCy** for natural language processing and **dateparser** for robust date recognition.

## Features

- Extracts events and dates from text documents.
- Handles multiple dates in a sentence.
- Detects date granularity: year-only, month+year, or full date.
- Handles date ranges.
- Sorts events chronologically.
- Visualizes events in an interactive timeline.
